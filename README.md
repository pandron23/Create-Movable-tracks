# Movable Tracks
## is a Create add-on that brings a new level of rail engineering to the game by allowing train tracks to become part of moving contraptions!
### Source code on [GitHub](https://github.com/Fortsumter01/Movable-Tracks)
### =======
Instead of rails only serving as fixed infrastructure, this mod opens the door to...
- dynamic rail systems 
- more advanced transportation builds
- a much more immersive mechanical experience for Create players who want to push their builds further.

## Origins
This mod was **originally made exclusively for the [Create VC](https://www.createvc.org/) 
server,**
![Shameless CreateVC advertising ploy🤷‍♀️😅](https://cdn.modrinth.com/data/cached_images/942efe51c4cdc4efec6531756e292fffddf88356_0.webp)
giving that community the chance to be the first to experience and showcase something **truly** original. It was built to give the server its own identity, its own special mechanic, and a feature players could **proudly** call theirs. 
But even though Movable Tracks began as something created specifically for that server, it is now being released publicly so that more players, builders, and engineers can enjoy it too. What started as a custom project for one community can now become a source of creativity for many others.

## Inspiration
**Movable Tracks** is for players who want **more** than static railways.
It is for builders who want their worlds to feel alive, for engineers who want to experiment with new mechanics, and for Create fans who love turning ambitious ideas into real working contraptions.
By making rail systems feel more dynamic and interactive, the mod helps transform tracks from simple transport lines into a **true centerpiece of mechanical creativity.**

## Practicality
Whether you want to build impressive transport systems, design memorable contraptions, or bring a completely new style of engineering to your world, Movable Tracks is made to inspire that next step. It began as something special for a single server, and now it is being shared so everyone can take part in that same excitement.

## Features
- Adds support for dynamic, movable rail-based creativity
- Expands what players can do with Create train systems
- Encourages larger, more impressive engineering builds
- Easily optimize the way you move your trains up and down
- Originally created for the Create VC server
Now released publicly for the wider Create community

Installation information
=======

This template repository can be directly cloned to get you started with a new
mod. Simply create a new repository cloned from this one, by following the
instructions provided by [GitHub](https://docs.github.com/en/repositories/creating-and-managing-repositories/creating-a-repository-from-a-template).

Once you have your clone, simply open the repository in the IDE of your choice. The usual recommendation for an IDE is either IntelliJ IDEA or Eclipse.

If at any point you are missing libraries in your IDE, or you've run into problems you can
run `gradlew --refresh-dependencies` to refresh the local cache. `gradlew clean` to reset everything 
{this does not affect your code} and then start the process again.
